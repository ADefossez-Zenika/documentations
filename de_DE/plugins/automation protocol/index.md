Plugin zur Unterstützung der Odace SFSP-Reihe (drahtlos ohne Akku) Schneider to Jeedom. Es ermöglicht die Information der Schalter (Fensterläden / Szenen / Einzel / Doppel). Es ermöglicht die Steuerung der Aktuatoren (Verschluss / dcl). Außerdem können Sie Szenen und Gruppen verwenden.Plugin für das SicherheitsmanagementPlugin, das Unterstützung für bestimmte Broadlink-Produkte wie (RM, A1 usw.) hinzufügt Plugin für Deconz (conBee) Plugin, das JeedomPlugin Unterstützung für das Edisio-Protokoll zur Steuerung des IPX 800Plugin für die Verwendung des Legrand F454Plugin-Moduls hinzufügt Hinzufügen von Unterstützung für EnOcean zu JeedomPlugin, wodurch die Verwaltung des Z-WAVEPlugin-Protokolls ermöglicht wird. Hinzufügen von Unterstützung für den RfPlayer-Schlüssel von Ziblue zu JeedomPlugin Hinzufügen von Unterstützung für den RfPlayer-Schlüssel von Ziblue zu Jeedom. Dieses Plugin wird eventuell das erste Plugin ersetzen. Es wird allen Besitzern des ersten Plugins zur Verfügung gestellt. Dadurch kann jeder ruhig migrieren. Da sich die Philosophie des Plugins geändert hat und auch die IDs, müssen Sie Ihre Geräte in dieses neue Plugin aufnehmen.Plugin zur Verwendung des RFXcom USB. Ermöglicht die Steuerung von 433-MHz-Hausautomationsgeräten (Chacon, Oregon, La Crosse, X10, DI-O, Blyss).
